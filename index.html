What is this?

This page is an index showing comparison heatmaps between the Book of Mormon and other books.

These heatmaps show how books are internally related, i.e. by breaking books into 1000-word segments, we can cross-compare the segments using a language model to show which segments are more related to one another. With this data, it's possible to look for internal relationships within a book, as well as relationships between two books.
